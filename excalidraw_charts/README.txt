Excalidraw 流程图文件说明
==========================

所有文件均可直接导入 https://excalidraw.com/ 查看和编辑。

文件列表
--------

flowchart_1.json
  对应论文图：图1-1 系统总体架构图
  内容：体现离线检测（OpenPCDet/PointPillar）与实时感知（RANSAC/DBSCAN）两条分支，
        最终汇聚到统一坐标系融合与 Open3D 可视化输出的完整系统架构。

flowchart_2.json
  对应论文图：图2-1 nuScenes Mini 数据集标注流程图
  内容：说明系统所使用模型依赖的数据集标注流程，包括多传感器采集、场景预处理、
        三维目标框标注、类别标签标注、质量检查与数据集生成。

flowchart_3.json
  对应论文图：图2-2 点云数据预处理流程图
  内容：展示点云从原始输入到预处理完成的完整流程，包括格式解析、噪声剔除、
        坐标归一化、体素下采样等步骤。

flowchart_4.json
  对应论文图：图3-1 系统算法整体流程图
  内容：以横向主流程展示系统算法链：点云输入 → 离线检测 → 实时分割 → 结果融合 → Open3D 输出。

flowchart_5.json
  对应论文图：图3-2 实时分割处理流程图
  内容：展示实时几何分割算法的完整步骤，包括点云采集、预处理、RANSAC 地面分割、
        非地面点提取、DBSCAN 聚类、障碍物边界框生成和结果输出。

flowchart_6.json
  对应论文图：图4-1 RealSense 实时点云获取流程图
  内容：展示 Intel RealSense 相机从启动到输出三维点云的数据流过程，
        包括深度图采集、图像对齐、深度去投影和实时点云生成。

导入方式
--------
1. 打开 https://excalidraw.com/
2. 点击左上角菜单（三条横线图标）
3. 选择 "Open" → "Open from file"
4. 选择对应的 .json 文件即可导入

导出为图片
----------
导入后，选中全部元素（Ctrl+A），点击菜单 → Export → 选择 PNG 或 SVG 格式导出，
建议勾选 "Background" 设为白色，适合论文插图使用。
